# GBASIC
GBASIC is a BASIC interpreter written in Go. Its implementation is loosely based on the Monkey interpreter from the book "Writing an Interpreter in Go" by Thorsten Ball. 

Currently, GBASIC is a work in progress. The goal for V1.0 is to cover all the TinyBASIC features. Ultimately, I'd like to implement the full GW-BASIC feature set.


## Personal motivation
My grandpa is a tech enthusiast and he used to write programs in GW-BASIC and Assembly both for work and leisure. Recently, he told me that he would like to play around with GW-BASIC again.

I thought it would be a fun project to write a BASIC interpreter in Go that he could use to run his old programs and write new ones.
